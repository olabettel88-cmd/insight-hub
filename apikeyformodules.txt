LeakOsint Search
Advanced breach database search with intelligent query parsing. Supports multi-word searches, automatic rearrangement, and precise quoted searches. Cost: (5 + sqrt(Limit * Complexity)) / 5000 USD per request.

Query complexity depends on word count: 1 word=1, 2 words=5, 3 words=16, 3+ words=40. Use double quotes for exact phrase matching. Dates, strings <4 chars, and numbers <6 digits are excluded from word count.

POST
https://www.breach.rip/api/leakosint-api
Parameters:
request
(string or array)
required
Search query or array of queries (e.g., 'john@example.com' or ['john@example.com', 'Jane Doe'])

limit
(integer)
Search limit (default: 100, range: 100-10,000). Determines number of returned results and search range.

lang
(string)
Language code for results (default: 'en'). Available: en, ru

Example Request:

curl -X POST "https://www.breach.rip/api/leakosint-api" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{"request": "value", "limit": "value", "lang": "value"}'
Response Format:
{
  "success": true,
  "data": {
    // Response data varies by endpoint
  },
  "timestamp": "2025-01-20T12:00:00.000Z"
